# Type Defender

## 구현 현황
- Game Over 성적표를 구현했습니다.
### index.html
- 성적표의 뼈대 컨테이너를 추가했습니다.
- 결과 화면에 뜨는 점수, 콤보, 레벨 등을 표시하는 영역을 추가했습니다.
- 다시 수강, 도전과제, 순위표 버튼을 추가했습니다. 도전과제, 순위표 버튼은 아직 로직을 구현하지 않았습니다.

### game.js
- 전체 정확도 계산을 위해 maxCombo, totalWordAttempts, successWords 등의 변수를 추가했습니다.
- handleSuccess()에서 단어를 맞출 때마다 successWords를 증가시키고 현재 combo와 비교하여 maxCombo를 갱신합니다.
- 게임 종료 방식을 결과창 UI로 연결되는 방식으로 변경했습니다.

### ui.js
- 결과 화면을 제어하는 핵심 로직을 추가했습니다. 
- showGameOver 함수를 추가하여 전체 정확도를 계산합니다.
    - 최고 점수를 판별하여 신기록을 알리는 기능을 추가했습니다.
    - 점수와 정확도 기준에 따라 정해진 등급 스탬프를 결정하는 기능을 추가했습니다.
        - A+ 등급: 점수가 1,000점 이상이면서 동시에 정확도가 95% 이상일 때
        - A 등급: 점수가 700점 이상일 때 (A+ 조건을 만족하지 못한 경우)
        - B 등급: 점수가 400점 이상일 때
        - C 등급: 점수가 150점 이상일 때
        - F 등급: 점수가 150점 미만일 때 (위의 모든 조건에 도달하지 못했을 때)
    - 결과 화면에 들어갈 점수, 콤보, 레벨 등을 화면에 나타나도록 active 클래스를 추가했습니다.
- hideGameOver 함수를 추가하여 결과 화면을 숨기는 기능을 추가했습니다.

### main.js
- 재도전(다시 수강) 로직을 위해 초기화 파트를 수정했습니다.
- 게임 객체 생성, UI 초기화 로직을 initGame 함수로 분리했습니다.
- UI 숨기기, initGame 함수를 묶는 restartGame 함수를 추가했습니다.
- 다시 수강 버튼을 누르면 게임이 재시작되는 기능을 구현했습니다.
