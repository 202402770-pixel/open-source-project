# Type Defender

교수님의 공격 단어를 빠르게 입력해서 막아내는 타이핑 디펜스 게임입니다.

## 현재 구현 범위

### W1 UI Components

전재형 담당 UI 컴포넌트 데모 페이지를 구현했습니다.

구현한 컴포넌트는 다음과 같습니다.

- Button
- Toggle
- Word Chip
- HUD Pill
- Professor + Desk 일러스트

## 파일 구조

- index.html: 프로젝트 첫 화면
- components.html: W1 UI 컴포넌트 데모 페이지
- css/style.css: 전체 스타일 및 컴포넌트 디자인
- js/ui.js: UI 상호작용 기능

## 실행 방법

index.html을 브라우저에서 열면 첫 화면을 확인할 수 있습니다.

components.html에서는 W1 UI 컴포넌트를 단독으로 확인할 수 있습니다.

## 추가 구현 내용

Toggle 컴포넌트는 JavaScript를 통해 클릭 시 선택 상태가 변경되도록 구현했습니다.
